# Prompt: questions_for_team.md

```text
Use #docs/transcript_context_report.md as the primary source.

Create a markdown file called:
questions_for_team.md

Purpose:
Turn the highest-value unresolved gaps into targeted questions for teammates.

For each question, include:
- Question
- Why I am asking it
- Current hypothesis
- Best person or role to ask
- Related component or term

Rules:
- Prefer specific, answerable questions.
- Avoid vague "can you explain X?" phrasing.
- Group questions by component or topic.
- Include only the highest-value questions.

Write the file in markdown.
```
