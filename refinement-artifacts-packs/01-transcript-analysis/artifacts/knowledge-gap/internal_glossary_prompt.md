# Prompt: internal_glossary.md

```text
Use #docs/transcript_context_report.md as the primary source.

Create a markdown file called:
internal_glossary.md

Purpose:
Collect important terms, acronyms, services, jobs, queues, and shorthand that seem relevant but were insufficiently explained.

For each glossary item, include:
- Term
- Likely meaning or role
- Why it matters
- Status (confirmed / inferred / unclear)
- Related components
- What still needs confirmation

Rules:
- Only include terms that appear important to understanding the work.
- Clearly label uncertainty.
- Do not pretend an unexplained term is fully understood.

Write the file in markdown.
```
