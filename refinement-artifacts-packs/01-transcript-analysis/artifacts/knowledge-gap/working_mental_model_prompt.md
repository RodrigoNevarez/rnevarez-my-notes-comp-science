# Prompt: working_mental_model.md

```text
Use #docs/transcript_context_report.md as the primary source.

Create a markdown file called:
working_mental_model.md

Purpose:
Write a plain-English explanation of how the system seems to work based on the report.

Structure:
- What the system appears to be doing
- The most important components
- How they seem to connect
- What still feels uncertain
- What I currently believe is true

Rules:
- Keep it readable and human-centered.
- This is a mental model, not a formal spec.
- Clearly mark uncertainty where needed.
- Avoid jargon where possible.

Write the file in markdown.
```
